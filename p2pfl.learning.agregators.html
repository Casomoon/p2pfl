class p2pfl.learning.agregators.agregator.Agregator(node_name='unknown')[source]#

Bases: Thread, Observable

Class to manage the agregation of models. Its a thread so, agregation will be done in background if all models was added or timeouts have gone. Also its a observable so, it will notify the node when the agregation was done.

Parameters:

node_name – (str): String with the name of the node.
