class p2pfl.learning.pytorch.logger.FederatedTensorboardLogger(dir, name=None, version=0, **kwargs)[source]#

Bases: LightningLoggerBase

Logger for PyTorch Lightning in federated learning. It is made to save the information of the different trainings (in the different rounds) in the same graph.

End of round determined with finalize_round

Parameters:

  • dir (str) – Directory where the logs will be saved.

  • name (str) – Name of the node.

  • version (int) – Version of the experiment.
